Automating Employee Data
📌 Project Overview

This project focuses on automating the management of employee data using ServiceNow. It eliminates manual data entry by using Import Sets and Transform Maps to efficiently import and store employee records in the User table (sys_user).

The system improves data accuracy, reduces redundancy, and ensures efficient handling of employee information.

🎯 Objectives
Automate employee data import
Reduce manual errors
Improve data consistency
Store employee records in ServiceNow
Simplify data management process

🛠️ Technologies Used
ServiceNow Platform
Import Sets
Transform Maps
JavaScript (optional)
CSV / Excel
